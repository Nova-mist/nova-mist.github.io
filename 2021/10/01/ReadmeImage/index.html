python为图片规范命名

发表于 2021-10-01

功能

将单张图片拖入程序，使得图片以规范的格式（blog-image-{日期}-{时间戳}）重命名。

Code

python "rename.py" %*
@REM pause
import os,sys,time
file_path = sys.argv[1]
# add strptime
t = int(time.time())
# get suffix
suffix = file_path.split('.')[1]
new_name = 'blog-image-' + str(t) + '.' + suffix
new_path = os.path.join('./',new_name)

# print(file_path)
# print(new_path)
os.rename(file_path, new_path)
