python面向对象复习

类相当于模板。

对象是根据类创建的实例。

类中定义方法（method），区别于普通的函数（function），两者是不同的类型。

类中的方法是绑定方法，会具体绑定到某一类的实例。当方法被调用时，实例对象会作为第一个参数(self)，被传入到方法中。

class Dog:
    def bark(self):
        print("wawa ~")


if __name__ == '__main__':
    beila = Dog()
    beila.bark()

也可以直接通过dot来使用类的属性和方法。

class Dog:
    color = "black"
    age = 12

    def bark(self):
        print("wawa ~")


if __name__ == '__main__':
    print(Dog.color)
    print(Dog.age)
    Dog.bark("xxx")  # Expected type 'Dog', got 'str' instead

__init__() 方法在类被实例化的时候被调用，构造函数。

传入的参数要与方法的参数相对应，也可以使用默认值。

class Dog:
    color = "black"
    age = 12

    def __init__(self, name="default", color="unknown"):
        self.name = name
        self.color = color

    def bark(self):
        print("wawa ~")


if __name__ == '__main__':
    dog1 = Dog()
    dog2 = Dog("可丽")
    dog3 = Dog('贝拉',"black")

    print(dog1.name, dog1.color)
    print(dog2.name, dog2.color)
    print(dog3.name, dog3.color)

对象方法

Objects can also contain methods. Methods in objects are functions that belong to the object.

class Dog:

三种方法

类中可以定义三种方法：

  • 常规实例方法
  • @classmethod类方法
  • @staticmethod静态方法

类方法

把cls（代表当前类）作为第一个参数，表示该方法与类绑定，可以调用专属于类的属性。

可以直接调用，也可以实例化。

静态方法

不强制传入self，不能直接访问类或实例的变量（需要通过类名访问），相当于定义在类外的普通函数。

无需实例化，也可以实例化后调用。

🟠都可以访问具体某个对象的属性和方法。

cls相当于类的别名

直接通过 类名.属性 的方式访问不到实例化才产生的属性，只可以访问类特有的属性和静态方法。

class Dog:
    color = "black"
    age = 12

    def __init__(self, name="default", color="unknown"):
        self.name = name
        self.color = color

    def bark(self):
        print(self.name,"wawa ~")

    def getName(self):
        return self.name

    @classmethod
    def slow_run(cls):
        print(cls.age, "age", cls().getName(), "run slowly.")
        print(Dog.age, "age", Dog().name, "run slowly.")

    @staticmethod
    def fast_run():
        print(Dog.age, "age", dog3.getName(), "run quickly.")
        print(dog2.age, "age", dog2.name, "run quickly.")
        Dog.stop()
        dog3.stop()

    @staticmethod
    def stop():
        print("stop!")

if __name__ == '__main__':
    dog1 = Dog()
    dog2 = Dog("可丽")
    dog3 = Dog('贝拉',"black")

    dog2.slow_run()
    dog2.fast_run()
