问题

在之前配好了命令行的代理之后，使用 curl 命令测试能够访问 google。

但是在使用 choco 命令时候又出现了错误。

Error retrieving packages from source 'https://chocolatey.org/api/v2/':
 无法连接到远程服务器

研究了一下，发现还是需要为 choco 命令单独设置代理。

设置 choco 代理

choco config set proxy http://127.0.0.1:[port]
choco config unset proxy # 重置
