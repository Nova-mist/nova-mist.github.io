Bresenham 画线法

判别式：

s<t 选择S点；s-t>=0 选择T点

s-t=2m(xi+1)+2b-2yi-1

迭代公式：

di+1 = di+2(▲y-▲x) di>=0

​ = di+2▲y di<0

起点判别式：

di = 2▲y - ▲x

void Cline::swap_value(int* a, int* b) {
	int temp = *a;
	*a = *b;
	*b = temp;
}
void Cline::Bresenhamline(CDC* p,int x1, int y1, int x2, int y2, int color)
{
	p->SetPixel(x1, y1, color);
	int dx = abs(x2 - x1);
	int dy = abs(y2 - y1);
	if (dx == 0 && dy == 0) {
		return;
	}
	int flag = 0;
	// 变换斜率到[0,1]
	if (dx < dy) {
		flag = 1;
		swap_value(&x1, &y1);
		swap_value(&x2, &y2);
		swap_value(&dx, &dy);
	}
    // 递增方向
	int tx = (x2 - x1) > 0 ? 1 : -1;
	int ty = (y2 - y1) > 0 ? 1 : -1;
    // 路径上的点
	int curx = x1;
	int cury = y1;
    // 判别式
	int dS = 2 * dy;
	int dT = 2 * (dy - dx);
	int d = dS - dx;
	while (curx != x1) {
		if (d < 0)
			d += dS;
		else
		{
			cury += ty;
			d += dT;
		}
		if (flag)
			p->SetPixel(cury, curx, color);
		else
			p->SetPixel(curx, cury, color);
		curx += tx;
	}
}

二维图形生成

直线生成算法

图元：点、直线、圆弧、多边形、字体符号、位图，这些基本图形元素能构成任何复杂的图形。

  • 三维图形的显示通过投影转化成二维图形的显示。
  • 图元不可再分。
  • 图元的描述 = 坐标信息 + 属性信息（宽、型、色、图案）

直线生成算法：

  1. 数值微分法
  2. 逐点比较法
  3. 中点画线法
  4. Bresenham 算法

区域填充

区域：

  • 给定顶点序列定义的封闭多边形。
  • 由点阵方式描述的区域。
    • 由边界定义的区域，填充的颜色可以和边界上的相同，边界填充算法
    • 内定义区域，区域内与区域外颜色不同，漫水法

对多边形区域进行填充

  1. 直接对多边形做扫描转换处理
  2. 将多边形的边界先画出，将边界包围的内部当做边界定义区域进行漫水法填充

种子填充算法

四连通、八连通

递归、堆栈法

多边形包含性测试算法

  1. 射线法

    点P向任意方向发出一条射线，若与多边形交点的个数为奇数则在多边形内，偶数在多边形外部。

    P发出的射线正好通过多边形的顶点：如果与顶点连接的两条边全部位于射线一侧时记为两个交点，两侧记为一个交点。

  2. 弧长法

多边形扫描转换算法

与当前扫描线有交点的边——活动边

活动边信息 –> 活动边表 AET （边处于激活状态）

边表ET：对边结点实施有效的激活和取消激活操作需要一个存储桶表，每一个存储桶代表一条扫描线

图形几何变换的模式

  • 固定坐标系模式
    • 图形改变，坐标系不变
    • 先调用的变换先执行（在矩阵右边）
    • 每一次变换均可看成相对于原始坐标系执行的
  • 活动坐标系模式
    • 图形不变，坐标系改变
    • 先调用的变换后执行（在矩阵左边）
    • 每一次变换均可看成是在上一次变换形成的新坐标系中进行的
