学习的计划

发表于 2021-03-12 更新于 2022-02-21

今天看了两个关于编程语言选择的视频，很对应俺的想法。顺便俺来说一点点以后的规划。

  • 这么多编程语言，初学者应该学哪个？推荐什么书？Java、C、Python、C#……
  • C++、Go语言怎么样？40岁程序员聊聊编程的铁饭碗

2022 / 2 / 21 更新想法

也到了大三下的时候，不管是想要做的事情还是不得不做的事情都好似搭着传送带接连不断地到来。

现在的我已经过了那个什么都感兴趣什么都想学的阶段了，现实的真实让我不再幻想未来，眼下只想脚踏实地的去做那些应做之事。

我只是个普通人，能过安心的生活就是我的期望。

上半年应该就接着学 Java，然后刷刷算法和面经，粗略看看操作系统还有 Golang。

学习哪门语言

不光是对于哪门语言，可以说对于很多遇到的知识，俺以前都是觉得“应该很有意思那就都学一学吧”，也就在前几周俺还在看着 AE 的学习视频照猫画虎地摆弄着步骤。可是现在想来俺做的这些说实在并没有用处，俺以后也不会成为一个就随便剪片子、做特效的人，可能一些很有趣的 AE 小视频总是吸引着俺也想学一学，可是俺就算学了一些以后照着教程复制了出来，这又有什么用处呢，最难的是用自己的想法去创作，而俺这方面实在一般般，而且也许学着学着又被别的有意思的东西带走了。（说实话，那个 AE 视频俺也就只坚持看了两三天，接下里就搁置了）

想学任何东西，想做任何事情，什么都选，那你最终什么都不是。

Frank警告你们：没有100%全能的人。因此，你不能什么专业领域都学，你不能什么专业都通吃，你也不能什么都做。
作者：Frank-FuckPPT
https://www.bilibili.com/read/cv10166082
出处： bilibili

像这篇专栏里说的一样，俺已经大二下了，已经无法承载”什么都想学“这个沉重的想法，当然俺已经在用排除法选择自己要学的东西和未来的职业了。

好了，至于要学习哪门语言，那当然是 Java 了。

当然 java \ python \ c++ \ c# \ python \ js \ go \ php 俺都多少看过、”临摹“过一些代码，可真的要说真正记住了什么、写出了什么自己的东西，俺是没有什么的。也许就像视频里说的”都会写 ‘helloworld’ 和循环”的程度吧。🤓

当你深入地掌握了一门语言后，其他语言也就触类旁通了。

确实，因为最开始学校学的 c/c++ ，在尝试别的语言地时候俺也能比较轻松地看懂内容。但看懂的这也也都是共通的皮毛罢了，深入的语言特色是学不到的。

所以这几天就开始学 Java 吧，还有要开始刷算法题了，然后在不久的将来开始做做一些小项目。

纠结之事

俺现在并不能准确地说出以后想干哪一行，所以就用排除法来暂做决定把。

先留下一些悬而未决的东西，或许下次更新就能解决了。
