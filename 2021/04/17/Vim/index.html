Vim 笔记

发表于 2021-04-17 更新于 2022-02-20

Vim 很好用，但我选择 IDE。 ♥

模式

操作符-次数-动作(motion)

d 3 w

打开文件

vi 文件名 +行数 # 跳到某行
vi 文件名 + # 跳转到末尾
ls -la

末行模式命令

:w
:w nihao.txt # 另存
:q
:!q
:wq = :x # 保存并退出

移动命令

在动作前输入数字n会使命令重复执行n次

w/b # 后/前移动一个单词 W/B 忽略标点
e # 到当前单词末尾
0 ^ $ #  行首 本行第一个非空字符 行尾
gg/G # 文件头部/尾部
num gg/G = :num # 跳转某行
ctrl b/f # 移动屏幕 上/下
H/M/L # 屏幕顶/中/底
{/} # 上/下一段落
% # 括号匹配
f{char}/F{char} #跳到下一个字符为 char 的位置/跳到上一个字符为 char 的位置    `;` 重复这个命令
Ctrl-F/B 下上翻页
````

 

 

 


## 标记 & 跳转

 


``` bash
mx

`x # 返回标记

选中

v/V/ctrl+V # 可视模式/可视行模式/可视块模式（像鼠标）
ggVG # 选中所有
V [/] # 选中一段
>>/<< # 缩排
V </> # 选中模式下调整缩进

恢复/删除

许多改变文本的命令都由一个操作符和一个动作构成。

u/ctrl+r # 撤销/恢复
U # 撤销整行的修改
x
d # 结合移动命令 dw d0 d{/} d4G d‘a删除标记
dd # 删除所在行 num+dd删除多行
D # 删至行尾
diw # 删除当前光标所在的单词
dt{char} # 从当前位置删除到某个字符前为止  `.` 重复这一步骤
3x
5dd
dW # 下一个单词起始
d$ # 到末尾
d0 # 当前行启示
d^ # 下一个非空字符
dG # 到行尾
d20G # 当前行到文件第20
de

文本对象

d + iw # 命令 + 对象
aw # 当前单词以及空格
i( # 括号内的内容
a( # 括号以及括号内的内容
 
yiw # 来复制光标下的单词：

选择性保存命令v

第一步：按v进入可视模式，选择要复制的内容
第二部：按：，屏幕底部出现:’<, ‘>，此时输入w filename即可将选中的内容复制到文件filename中去

更改操作符

更改类操作符的工作方式跟删除类是一致的。操作格式是：
c [number] motion
动作参数(motion)也是一样的，比如 w 代表单词，$代表行末等等。

复制/粘贴/剪切

y 移动命令 p
yy # 复制一行
n yy # 复制多行

p 下一行粘贴
P 本行粘贴
 
 
shift + v # 进入选择模式
d y p # 剪切 复制 粘贴
n + 命令 # 多次
 
yy
5yy
yW # 到下一单词的开头
y$
y0
y^
yG
y20G

J 命令

替换/查找

f 行内搜索

r/R # 当前字符/替换模式
. # 重复上一次操作
#更改文本直到一个单词的末尾 - ce
?str # 逆向查找
/str n/N # 查找 下一个/上一个
*/# #光标移到当前str 快速匹配 光标向上/
:%s/ / /g #查找替换
# 范围 % 头到尾 也可指定 1,5 1,$
# s 替换操作
:%s/old_str/new_str/g # 全局
:s/old_str/new_str/g # 先选中文本
:%s/old_str/new_str/gc # 确认替换
# 选项y(es)/n(o)/a(ll)/q(uit)/l(ast最后一个替换返回行首)/^E/^Y上滚/下滚

编辑多个文件

vi file1 file2 fil3
:n # 切换下一个文件
:N # 上一个
:buffers # 文件列表
:buffer 1 :buffer 2 # 切换
:e hello.txt # 载入更多文件
# 文件间复制
:r foo.txt # 插入整个文件

插入命令

i/I/a/A/o/O # 当前/行首/当前字符后/行末/后空行/前空行
I == ^i
A == $a
O == ko

多行代码注释

yy5p # 复制多次本行
num i `*` Esc # 插入多个字符 进入编辑模式下不要输入数字 会重复多次
^/0 ctrl V j I # Esc # 移动到行首 可视块模式 下移 行首插入`#` 退出

末行命令补充

:e :e. :e 文件名 # 文件名字 查看目录 打开文件

设置类命令的选项

:set ic - 搜索字符串时忽略大小写
:set noic - 禁用忽略大小写
:set hls - 高亮搜索结果（或者使用:set hlsearch）
:set nohlsearch
