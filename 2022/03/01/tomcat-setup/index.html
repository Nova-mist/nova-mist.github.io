安装Tomcat和相关问题

安装方法

官网下载地址

  • 直接下载 exe 安装程序，不用配置环境。
  • 下载压缩包解压后配置环境变量。

JAVA_HOME C:\Program Files\Java\jdk1.8.0_301

CATALINA_HOME D:\Program Files\apache-tomcat-9.0.58

IDE 中配置

注意要选择 Tomcat Server 的 Local 服务，而不是 TomEE Server。

调试设置中的应用上下文路径要与网址的路径一直，最好都设为空。

设置动态刷新页面

startup.bat 乱码问题

修改 D:\Program Files\apache-tomcat-9.0.58\conf\logging.properties 中的编码格式：

java.util.logging.ConsoleHandler.level = FINE

java.util.logging.ConsoleHandler.formatter = org.apache.juli.OneLineFormatter

java.util.logging.ConsoleHandler.encoding = GBK

端口占用问题

Error running ‘Tomcat 9.0.58’: Address localhost:8080 is already in use

netstat -aon | findstr '[port_number]'
tasklist | findstr '[PID]'

发现是因为运行的 IPFS 占用了8080端口，修改后问题解决。
