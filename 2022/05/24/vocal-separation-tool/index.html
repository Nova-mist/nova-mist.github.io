vocal-separation-tool

发表于 2022-05-24

准备

  • 使用BBDown下载视频。
  • 重命名
  • 提取片段
bbdown --audio-only https://www.bilibili.com/bangumi/play/ep477128 
Rename-Item -Path '.\测不准的阿波连同学`[P7`]7 这是艺术了吧？.mp4' 'p7_audio.mp4'
ffmpeg -ss 00:14:17 -to 00:14:50 -i p7_audio.mp4 -c:v copy -c:a copy p7_audio_v2.mp4

✅p7_audio_v2.mp4

转换成WAV格式。

ffmpeg -i .\p7_audio_v2.mp4 -ac 2 -f wav p7_audio_v2.wav

提取人声

  • LALAL.AI: 100% AI-Powered Vocal and Instrumental Tracks Remover
    免费版只支持上传 wav 格式音频，有使用10min总时长。十几秒的需求就用它。

  • deezer/spleeter: Deezer source separation library including pretrained models. (github.com)
    ❌下面的命令无法安装
    🟢使用 spleeterGUI
    一 键 完 美 提 取 人 声_哔哩哔哩_bilibili 用百度网盘下吧。。

    mkdir MyApp
cd MyApp
py -m venv env
.\env\Scripts\activate # 运行激活脚本就会进入虚拟环境
deactivate # 离开虚拟环境
# 接下来就可以在一个干净的环境安装包了
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
pip config set install.trusted-host mirrors.aliyun.com
py -m pip install spleeter

# download an example audio file (if you don't have wget, use another tool for downloading)
wget https://github.com/deezer/spleeter/raw/master/audio_example.mp3
# separate the example audio into two components
spleeter separate -p spleeter:2stems -o output audio_example.mp3

  • 🟠也可以去找多声道片源用pr提取
