了解下Disco Diffusion

发表于 2022-05-07

Disco Diffusion

https://weibo.com/1757693565/Lnu7RBqzI

Settings

Basic Settings

参数：

  • step 决定生成图片的精细程度
  • width_height 长宽需要是64的倍数
    • [1280, 768]
    • [800, 800]
    • [1280, 512] 电影风格

Init Settings

  • init_image 路径，使用图片作为参考模板
  • init_scale 数值越大，与参考图片越相似
  • skip_steps 设置前多少step使用参考图片

Extra Settings

  • intermediate_saves 设置保存生成中的关键帧（百分率）

Prompts

输入描述性的语句。

主体，构成，作者风格，参考网站，补充细节。

Diffuse

  • display_rate 每隔多少step展示生成过程中的关键帧
  • n_batches 最终生成的图片数（使用相同的描述）

Run

Run all

Run anyway

Use Google Drive.

resume_run

Error

1.3 No CUDA GPUs.

可用资源用尽，需要等待冷却。登录不用也会消耗

CUDA out of memory.

图片分辨率过大 / 运行时间过长。

终止会话，重新加载。

1.3 加载错误导致 2 / 4 出现错误。

清空云端，重新加载。
