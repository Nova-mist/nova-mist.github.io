试试PowerShell

发表于 2022-07-11 更新于 2022-08-04

试试PowerShell

  • PowerShell/PowerShell: PowerShell for every system! (github.com)
  • What is PowerShell? - PowerShell | Microsoft Docs
  1. PowerShell是支持全平台使用的。
  2. Windows上内置的是v5.1版本，最新的v7版本需要自行安装。

配置环境

  • 修改Windows Terminal配置，排列、主题。
  • 配置 oh-my-posh —— 提示符个性化工具。

oh-my-posh

参考：

  • Oh My Posh：全平台终端提示符个性化工具 - 少数派 (sspai.com)
  • Introduction | Oh My Posh

使用搭配：Windows Terminal + oh-my-posh + PowerShell Core
💥【终端】+【个性化配置工具】+【shell】

安装

winget install oh-my-posh
winget upgrade oh-my-posh

配置主题

在 主题页面 查看内置的主题样式，对应的配置文件的路径为 ~\AppData\Local\Programs\oh-my-posh\themes\xxx.omp.json

使用PowerShell查看配置文件的路径 $Profile ，创建并编辑该 .ps1 文件：

oh-my-posh init pwsh --config ~/AppData/Local/Programs/oh-my-posh/themes/cloud-native-azure.omp.json | Invoke-ExpressionR

修改字体

下载 Nerd Fonts系列字体 并安装，在Windows Terminal中选择。🟠（全选-右键-为所有用户安装）

重启Windows Terminal查看配置完成的效果。

💥修改vscode中的终端字体

常用命令

$PSversionTable # 查看版本
$Profile # 配置文件的路径
# 重命名
Rename-Item -Path '.\测不准的阿波连同学`[P7`]7 这是艺术了吧？.mp4' 'p7_audio.mp4'
# 删除目录下的特定文件 正则表达式 Get-ChildItem选项 -Recurse 获取子目录
Get-ChildItem $Path | Where{$_.Name -Match "<RegEx Pattern>"} | Remove-Item
