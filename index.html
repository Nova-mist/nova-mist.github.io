解决 Typora 编辑为知笔记频繁失焦问题

发表于 2021-05-10

最近将 Typora 更新到了 0.10 版本，却发现编辑以前完美搭配的为知笔记内容时频繁失去窗口焦点，目测应该是新版 Typora 在自动保存后会切换焦点到同样进行保存行为的为知笔记上。

解决方法：在Typora for windows — dev version release下载，回滚历史版本（Windows），不用卸载，只需覆盖安装。

其他端版本亦如此点击 Change Log 进入历史发布版本界面。

PicGo设置GitHub图床

发表于 2021-03-31

参考

  • .net core3.0上传文件出现404_PicGo+Github,实现Typora自动上传图片文件
  • PicGo+GitHub使用
  • 图床搭配 PicGo：打造高效的图片处理工作流

准备

  1. 注册一个 163 邮箱
  2. 用此邮箱注册一个专门存储图片的 GitHub 账号
  3. 生成仓库 用户名/仓库名
  4. 生成 token

全部勾选就好。

设置 PicGo

⚠ 注意：

  • 仓库名要写 用户名/仓库名 用户名是用来区分仓库的（用户名唯一）。
  • 分支名现在改成 main 了，不是原来的 master
  • 存储路径就是仓库中文件存储的路径，会自动生成文件夹。
  • 自定义域名用于图片加速 CDN 服务，一般不用填写，如果单纯想填就写 https://raw.githubusercontent.com/用户名/仓库名/main

如需 CDN 加速，jsdeliver 配置

  • 配置格式：https://cdn.jsdelivr.net/gh/用户名/仓库名
  • 举例：https://cdn.jsdelivr.net/gh/huayonglun/cdn_image001

就此就可以使用 GitHub 图床了。😸

Docker 学习

适用于 Linux 的 Windows 子系统安装指南 (Windows 10)

  • 按步骤安装 WSL 2

Docker 简介

😸俺的理解

Docker 就相当于一个轻量级的虚拟机。在不保存数据的前提下（可以保存），可以很轻松的学习 Linux 的各种操作，并且不用担心恢复配置的问题（关闭容器数据就清除了）。

在后续使用中可以使用 Docker 在服务器上快速部署服务，PHP MySQL Tomcat 等。

Docker 简单使用

Windows 10 安装

点击以下 [链接](https://desktop.docker.com/win/stable/Docker Desktop Installer.exe) 下载 Docker Desktop for Windows。

推荐使用 Windows Terminal 在终端使用 Docker。

Windows Terminal 主题美化

配置国内镜像加速

使用镜像

# 拉取镜像 docker pull --help
docker pull [选项] [Docker Registry 地址[:端口号]/]仓库名[:标签]
docker pull ubuntu:18.04 # latest
docker run -it --rm ubuntu:18.04 bash # 启动一个交互终端 退出后删除 /bin/bash
# cat /etc/os-release 查看信息
docker image ls # 列出镜像 参数 -a 查看所有镜像 参数 -q 镜像ID
docker image rm # 删除镜像 后加ID

容器命令

# 新建并启动
docker run ubuntu:18.04 /bin/echo 'Hello world' # 命令完成后就停止容器
docker run -it ubuntu:18.04 /bin/bash # 启动一个 bash 终端 可以交互
# exit Ctrl+d 退出终端
docker container start[restart][stop]
docker container ls [-a] # 查看列表
docker attach [id] # 进入容器
docker container rm [id] # 删除容器
docker container prune # 清理所有处于终止状态的容器
