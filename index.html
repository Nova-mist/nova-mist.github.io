把Line表情转换成微信表情

发表于 2022-07-13

把Line表情转换成微信表情

准备工作

  • 安装Chrome插件 line贴图打包
  • 在 line商店 找到想要的表情包并下载
  • 安装 image magick

转换格式

🟢对于静态图片，只需要使用Image magick的命令行。

# 新建文件夹
mkdir .\output
# 获取路径下所有表情文件
$files = Get-ChildItem .\
# 转换每一个表情
foreach ($file in $files) {magick "$file" "$file".Replace("png","gif")}
# 将所有转换好的gif图片移动到 output文件夹
Get-Item *.gif | Move-Item -Destination .\output

名字带有 key 的是小图，正常微信表情适合用大图。

🟠对于动态图片，因为是apng的格式，一般的转换传到微信会显示错误，只能使用一个日语的转换网站 Magical Converter 来转成gif。

  • 一般动图的fps设定为 0.02秒（50fps），每帧的播放时间。本着APNG的总播放时间不能超过4秒参考，自行调整。
  • 对于还是会出现闪出白色背景的动图
    • 可以将背景颜色设定为#ededed，这样在手机上不会看出闪动，但电脑上会显示闪几帧
    • 也可以取消背景透明的选项，再改背景颜色，这样在电脑上看背景是固定的颜色

测试的图片：

  • 静态：Office Worker ButterCat – LINE貼圖 | LINE STORE
  • 动态：星のカービィ まんまるスタンプ - LINE スタンプ | LINE STORE

试试PowerShell

发表于 2022-07-11

试试PowerShell

  • PowerShell/PowerShell: PowerShell for every system! (github.com)
  • What is PowerShell? - PowerShell | Microsoft Docs
  1. PowerShell是支持全平台使用的。
  2. Windows上内置的是v5.1版本，最新的v7版本需要自行安装。

配置环境

  • 修改Windows Terminal配置，排列、主题。
  • 配置 oh-my-posh —— 提示符个性化工具。

oh-my-posh

参考：

  • Oh My Posh：全平台终端提示符个性化工具 - 少数派 (sspai.com)
  • Introduction | Oh My Posh

使用搭配：Windows Terminal + oh-my-posh + PowerShell Core
💥【终端】+【个性化配置工具】+【shell】

安装

winget install oh-my-posh
winget upgrade oh-my-posh

配置主题

在 主题页面 查看内置的主题样式，对应的配置文件的路径为 ~\AppData\Local\Programs\oh-my-posh\themes\xxx.omp.json

使用PowerShell查看配置文件的路径 $Profile ，创建并编辑该 .ps1 文件：

oh-my-posh init pwsh --config ~/AppData/Local/Programs/oh-my-posh/themes/cloud-native-azure.omp.json | Invoke-ExpressionR

修改字体

下载 Nerd Fonts系列字体 并安装，在Windows Terminal中选择。🟠（全选-右键-为所有用户安装）

重启Windows Terminal查看配置完成的效果。

💥修改vscode中的终端字体

常用命令

$PSversionTable # 查看版本
$Profile # 配置文件的路径
# 重命名
Rename-Item -Path '.\测不准的阿波连同学`[P7`]7 这是艺术了吧？.mp4' 'p7_audio.mp4'
# 删除目录下的特定文件 正则表达式 Get-ChildItem选项 -Recurse 获取子目录
Get-ChildItem $Path | Where{$_.Name -Match "<RegEx Pattern>"} | Remove-Item

参考

  • regex - How can one delete files in a folder matching a regular expression using PowerShell? - Stack Overflow
