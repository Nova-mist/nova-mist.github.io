解决报错LF will be replaced by CRLF

发表于 2022-04-01

问题

每次 hexo deploy 发布博客的时候就会出现：

warning: LF will be replaced by CRLF

解决

在 .git 目录下的 config 文件中设置：

core.autocrlf = false

会关闭 git 默认的换行(LF)转换成回车和换行(CRLF)

🟠这种设置只适合纯 Windows 系统下的项目，由于只想应用于 hexo 博客，所以只修改了单个项目的配置。不建议使用全局配置 git config --global core.autocrlf false 。

参考

  • git如何避免”warning: LF will be replaced by CRLF“提示？ - 知乎
  • 通过阅读 git-config 文档理解 Git 如何使用autocrlf、safecrlf、eol和.gitattributes处理line-ending

mkdocs-learn

发表于 2022-04-01

Getting Started

Material for MkDocs - Material for MkDocs (squidfunk.github.io)

安装

  1. 使用 pip 安装：
pip install mkdocs-material
  1. 使用 git 安装并使用 pip 安装所需环境：
git clone https://github.com/squidfunk/mkdocs-material.git
pip install -e mkdocs-material

建站

初始化项目：

mkdocs new .

会产生如下目录结构：

.
├─ docs/
│  └─ index.md
└─ mkdocs.yml

基础配置

配置目录下的 mkdoc.yml

  1. 使用 pip 安装的配置：
theme:
  name: material
  1. 使用 git 安装的配置：
theme:
  name: null
  custom_dir: mkdocs-material/material

  # 404 page
  static_templates:
    - 404.html

  # Necessary for search to work properly
  include_search_page: false
  search_index_only: true

  # Default values, taken from mkdocs_theme.yml
  language: en
  font:
    text: Roboto
    code: Roboto Mono
  favicon: assets/favicon.png
  icon:
    logo: logo

tomcat-setup

发表于 2022-03-01

安装Tomcat和相关问题

安装方法

官网下载地址

  • 直接下载 exe 安装程序，不用配置环境。
  • 下载压缩包解压后配置环境变量。

JAVA_HOME C:\Program Files\Java\jdk1.8.0_301

CATALINA_HOME D:\Program Files\apache-tomcat-9.0.58

IDE 中配置

注意要选择 Tomcat Server 的 Local 服务，而不是 TomEE Server。

调试设置中的应用上下文路径要与网址的路径一直，最好都设为空。

设置动态刷新页面

startup.bat 乱码问题

修改 D:\Program Files\apache-tomcat-9.0.58\conf\logging.properties 中的编码格式：

java.util.logging.ConsoleHandler.level = FINE

java.util.logging.ConsoleHandler.formatter = org.apache.juli.OneLineFormatter

java.util.logging.ConsoleHandler.encoding = GBK

端口占用问题

Error running ‘Tomcat 9.0.58’: Address localhost:8080 is already in use

netstat -aon | findstr '[port_number]'
tasklist | findstr '[PID]'

发现是因为运行的 IPFS 占用了8080端口，修改后问题解决。

Reference

How to check which application is using which port - Veera / Blog (veerasundar.com)

How to Install Tomcat Server On Windows 10, 8 and 7 | ExamTray
