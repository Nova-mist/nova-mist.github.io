python面向对象复习

类相当于模板。

对象是根据类创建的实例。

类中定义方法（method），区别于普通的函数（function），两者是不同的类型。

类中的方法是绑定方法，会具体绑定到某一类的实例。当方法被调用时，实例对象会作为第一个参数(self)，被传入到方法中。

class Dog:
    def bark(self):
        print("wawa ~")


if __name__ == '__main__':
    beila = Dog()
    beila.bark()

也可以直接通过dot来使用类的属性和方法。

class Dog:
    color = "black"
    age = 12

    def bark(self):
        print("wawa ~")


if __name__ == '__main__':
    print(Dog.color)
    print(Dog.age)
    Dog.bark("xxx")  # Expected type 'Dog', got 'str' instead

__init__() 方法在类被实例化的时候被调用，构造函数。

传入的参数要与方法的参数相对应，也可以使用默认值。

class Dog:

三种方法

类中可以定义三种方法：

  • 常规实例方法
  • @classmethod类方法
  • @staticmethod静态方法

类方法

把cls（代表当前类）作为第一个参数，表示该方法与类绑定，可以调用专属于类的属性。

可以直接调用，也可以实例化。

静态方法

不强制传入self，不能直接访问类或实例的变量（需要通过类名访问），相当于定义在类外的普通函数。

无需实例化，也可以实例化后调用。

🟠都可以访问具体某个对象的属性和方法。

cls相当于类的别名

直接通过 类名.属性 的方式访问不到实例化才产生的属性，只可以访问类特有的属性和静态方法。

class Dog:
    color = "black"
    age = 12

    def __init__(self, name="default", color="unknown"):
        self.name = name
        self.color = color

    def bark(self):

    def getName(self):

    @classmethod
    def slow_run(cls):
        print(cls.age, "age", cls().getName(), "run slowly.")
        print(Dog.age, "age", Dog().name, "run slowly.")

    @staticmethod
    def fast_run():
        print(Dog.age, "age", dog3.getName(), "run quickly.")

        Dog.stop()
        dog3.stop()

    @staticmethod

        print("stop!")

Bresenham 画线法

判别式：

s<t 选择S点；s-t>=0 选择T点

s-t=2m(xi+1)+2b-2yi-1

迭代公式：

di+1 = di+2(▲y-▲x) di>=0

​ = di+2▲y di<0

起点判别式：

di = 2▲y - ▲x

void Cline::swap_value(int* a, int* b) {
	int temp = *a;
	*a = *b;
	*b = temp;
}
void Cline::Bresenhamline(CDC* p,int x1, int y1, int x2, int y2, int color)
{
	p->SetPixel(x1, y1, color);
	int dx = abs(x2 - x1);
	int dy = abs(y2 - y1);
	if (dx == 0 && dy == 0) {
		return;
	}
	int flag = 0;
	// 变换斜率到[0,1]
	if (dx < dy) {
		flag = 1;
		swap_value(&x1, &y1);
		swap_value(&x2, &y2);
		swap_value(&dx, &dy);
	}
    // 递增方向
	int tx = (x2 - x1) > 0 ? 1 : -1;
	int ty = (y2 - y1) > 0 ? 1 : -1;
    // 路径上的点
	int curx = x1;
	int cury = y1;
    // 判别式
	int dS = 2 * dy;
	int dT = 2 * (dy - dx);
	int d = dS - dx;
	while (curx != x1) {
		if (d < 0)
			d += dS;
		else
		{
			cury += ty;
			d += dT;
		}
		if (flag)
			p->SetPixel(cury, curx, color);
		else
			p->SetPixel(curx, cury, color);
		curx += tx;
	}
}

直线生成算法

图元：点、直线、圆弧、多边形、字体符号、位图，这些基本图形元素能构成任何复杂的图形。

  • 给定顶点序列定义的封闭多边形。

    • 内定义区域，区域内与区域外颜色不同，漫水法

对多边形区域进行填充
